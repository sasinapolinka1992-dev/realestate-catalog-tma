
# Инструкция по развертыванию на GitHub Pages

## Быстрый деплой
1. Создайте новый репозиторий на GitHub.
2. Загрузите файлы `index.html`, `index.tsx`, `App.tsx` и остальные в корень репозитория.
3. Перейдите в **Settings** -> **Pages**.
4. В разделе **Build and deployment** выберите ветку `main` и папку `/ (root)`.
5. Нажмите **Save**. Через 1-2 минуты ваше приложение будет доступно по адресу `https://<ваш-логин>.github.io/<имя-репозитория>/`.

## Интеграция с Telegram
1. Откройте [@BotFather](https://t.me/botfather).
2. Используйте команду `/newapp` для создания Mini App.
3. В качестве URL укажите вашу ссылку с GitHub Pages.

## Интеграция с VK
1. Зайдите на [vk.com/apps?act=manage](https://vk.com/apps?act=manage).
2. Создайте новое приложение (тип "Мини-приложение").
3. В разделе "Состояние" укажите URL GitHub Pages для мобильной и десктопной версии.

## Особенности Plan7
Приложение автоматически подтягивает интерактивный каталог по ссылке `https://plan7.ru/catalog/exp/index.php?zk=606&page=apa`. Вы можете изменить параметр `zk` в `constants.ts` для переключения на другой жилой комплекс.
